# dd-safe
A safer and easier way to use dd (written in POSIX sh).

## OS support
* :penguin: Linux
* :apple: macOS
* :smiling_imp: FreeBSD
* :dragon: DragonFly BSD
* :wine_glass: Windows via WSL (Windows subsystem for Linux)

### Coming soon...
* :blowfish: OpenBSD
* :triangular_flag_on_post: NetBSD
* :sunny: Solaris
